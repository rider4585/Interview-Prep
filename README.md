# Interview Prep

A React interview preparation app with topic tabs, difficulty levels, searchable Q&A, and coding challenges.

## Tech Stack

- React
- Vite

## Getting Started

Install dependencies:

```sh
npm install
```

Start the Vite development server:

```sh
npm run dev
```

Create a production build:

```sh
npm run build
```

Preview the production build locally:

```sh
npm run preview
```

## Project Structure

- `index.html` is the Vite HTML entry point.
- `src/main.jsx` mounts the React app.
- `src/App.jsx` contains the interview prep UI and question data.
- `vite.config.js` configures React for Vite.
